Layout - Dashboard
Overview
The dashboard page features a grid of widgets with a variable columns and rows. The position, type and size of widgets can be customized. Users 
are also able to create multiple dashboard views, which can easily be accessed via dropdown. 
The dashboard allows users to build custom view with the help of a variety of widgets. While the amount and type of widgets may differ depending on the application, 
the following basic constructions still apply.
