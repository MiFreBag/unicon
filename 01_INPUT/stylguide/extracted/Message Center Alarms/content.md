Layout - Events (Message Center)
Overview
Alerts
Core elements & construction
The “Events” (Message Center) section allows users to access various alerts, notifications and logs via a horizontal tab 
navigation. They displayed data is organized in tables which follow the general rules defined in the “Tables” page. 

Custom table layouts unique to the Events section can be found below.
The “Alerts” page provides information on all alerts that have been sent out by mail, SMS or other push methods for all devices in the tenant. It uses a 
24px spacing between columns. 

The number of pending alerts is shown in a notification bubble at the top right of the bell icon. The background color of the bubble corresponds to 
the most severe state that is currently active.
